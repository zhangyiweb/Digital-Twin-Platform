# 数字孪生平台 - 交互功能使用指南

## 🎮 当前可用功能

### 1. 3D视口交互

#### 相机控制
- **旋转视角**: 鼠标左键拖拽
- **平移视角**: 鼠标右键拖拽
- **缩放视角**: 鼠标滚轮滚动

#### 对象选择
- **点击选择**: 单击3D场景中的对象
- **取消选择**: 点击空白区域
- **Gizmo控制**: 选中对象后显示移动/旋转/缩放控制器

### 2. 工具切换

#### 工具栏按钮
点击顶部工具栏的工具按钮:
- 🔍 **选择** (Q键) - 选择对象模式
- ✋ **移动** (W键) - 移动选中对象
- 🔄 **旋转** (E键) - 旋转选中对象  
- 📏 **缩放** (R键) - 缩放选中对象

#### 显示控制
- **网格** (G键) - 切换地面网格显示
- **坐标轴** (H键) - 切换坐标轴辅助线显示

### 3. 场景树面板 (左侧)

#### 功能
- 显示场景中的所有对象
- 点击对象名称进行选择
- 展开/折叠层级结构
- 搜索对象 (搜索框)
- 查看对象类型图标:
  - 🔷 网格对象
  - 💡 灯光
  - 📷 相机
  - 📁 组

#### 交互
- 单击选择对象
- Ctrl+单击多选
- 点击眼睛图标切换可见性

### 4. 属性面板 (右侧)

#### 变换属性编辑
当选中对象时显示:

**位置 (Position)**
- X (红色) - 左右移动
- Y (绿色) - 上下移动
- Z (蓝色) - 前后移动

**旋转 (Rotation)**
- X - 绕X轴旋转
- Y - 绕Y轴旋转
- Z - 绕Z轴旋转

**缩放 (Scale)**
- X - X轴缩放
- Y - Y轴缩放
- Z - Z轴缩放

#### 信息区域
显示:
- 对象类型
- 对象ID
- 可见性状态

### 5. 底部状态栏

显示信息:
- 当前工具名称
- 选中对象名称
- 场景对象总数
- FPS帧率

## 🎯 使用流程示例

### 示例1: 移动对象
1. 按 **W** 键切换到移动工具
2. 点击场景中的立方体
3. 拖动出现的Gizmo箭头移动对象
4. 或在右侧属性面板输入精确坐标

### 示例2: 旋转对象
1. 按 **E** 键切换到旋转工具
2. 点击选中对象
3. 拖动Gizmo圆环旋转对象
4. 或在属性面板输入旋转角度

### 示例3: 查看对象信息
1. 点击场景中的任意对象
2. 左侧场景树高亮对应节点
3. 右侧属性面板显示详细信息
4. 底部状态栏显示对象名称

### 示例4: 切换显示
1. 按 **G** 键隐藏/显示网格
2. 按 **H** 键隐藏/显示坐标轴
3. 或点击工具栏的"网格"/"坐标轴"按钮

## 💡 开发提示

### 如何测试
1. 确保开发服务器运行: `npm run dev`
2. 打开浏览器: `http://localhost:5174/`
3. 尝试各种交互操作

### 如何调试
- 打开浏览器开发者工具 (F12)
- 查看Console日志
- 选中对象时会输出日志信息

### 文件位置
- 3D视口: `src/components/Viewport/EditorViewport.tsx`
- 场景树: `src/components/Panels/SceneTree.tsx`
- 属性面板: `src/components/Panels/PropertyPanel.tsx`
- 工具栏: `src/components/Toolbar/MainToolbar.tsx`
- 快捷键: `src/hooks/useKeyboardShortcuts.ts`
